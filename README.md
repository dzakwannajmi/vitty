# 🚀 Stellar Donation DApp

A simple decentralized donation application built on the Stellar Testnet using **Freighter Wallet**.  
This project was created as part of a **Soroban / Stellar workshop** to demonstrate wallet integration and basic transaction flow.

---

## ✨ Features

- 🔌 Connect wallet via Freighter
- 👤 Display user address & XLM balance
- 💸 Send XLM donation (Testnet)
- ⚡ Real-time balance refresh
- 🛡️ Error handling for wallet & network issues
